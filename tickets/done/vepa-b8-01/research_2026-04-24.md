# Research: VEPA-B8-01 - 4 Presets Implementation

## 1. Overview
The goal is to implement four new presets: `VOID_CORE`, `NEURAL_DRIFT`, `SOLAR_FLARE`, and `CRYSTAL_LATTICE` based on the `rrp_synthesis.md` document.

## 2. Technical Requirements
Each preset requires new logic in the `physics.worker.js` and other engines.

### 2.1 VOID_CORE
- **Accretion: Linear Evaporative Decay**: Requires a mass reduction law for large particles.
- **Genetic Hashing**: (Potentially in `lineageTracker.js`).
- **Gating**: Interaction-gated history recording.

### 2.2 NEURAL_DRIFT
- **Link Decay**: Dissolving connections when signals equalize.
- **Elastic Tethers**: Physical pull between connected particles.
- **Damping**: Signal loss over distance.
- **Parameters**: "Boredom" and "Distance Sensitivity".

### 2.3 SOLAR_FLARE
- **Radiation Grid**: 3D voxel grid for radiation values.
- **Sterilization**: Radiation disables reproduction/signaling.
- **Shadow Mapping**: Occlusion support in the radiation grid.

### 2.4 CRYSTAL_LATTICE
- **Conductive Flow**: Mass/Energy exchange on contact.
- **Elastic Rebound**: 100% kinetic energy reflection when saturated.
- **Capture**: Lock particles to grid when velocity is near zero.

## 3. Current State of `physics.worker.js`
- Handles basic gravity, collisions, and accretion.
- Biology loop handles life/death but no radiation or evaporation.
- STRIDE is 24.

## 4. Proposed DNA/Law Additions
- `pure.void`: Toggles evaporative decay.
- `pure.rad`: Toggles radiation grid.
- `pure.bond`: Toggles link/tether logic.
- `biol.rad_sensitivity`: DNA parameter for radiation impact.
