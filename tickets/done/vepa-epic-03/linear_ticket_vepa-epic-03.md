# Description

## Problem to solve
The simulation is complex and its emergent behaviors are often opaque to users. Tooltips are static and non-persistent, providing no deep context or inter-parameter relationship data.

## Solution
Implement a multi-layered Help & Encyclopaedia system (The Codex) and a real-time Insight Engine (EIE) that interprets simulation state into human-readable narratives and actionable suggestions.

## Implementation Batches
- **Codex UI:** Mobile-first persistent help panel and searchable encyclopaedia.
- **HELP_DB:** Layered documentation schema (Hint, Explanation, System, Advanced).
- **Insight Engine:** Rule-based detection of physical, biological, and swarm phenomena.
- **Storytelling:** Narrative log and auto-suggest experiment presets.
