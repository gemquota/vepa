# Description

## Problem to solve
Simulations reset on refresh, losing all discovered patterns and evolutionary history. The physics engine is static and cannot adapt to repeated emergent phenomena.

## Solution
Implement a persistent memory layer (The Universe Engine) that tracks lineage, records history on a scrub-able timeline, and abstracts repeated anomalies into new controllable "Emergent Parameters" (Meta-Laws).

## Implementation Batches
- **Timeline Engine:** Record/replay state with annotated explanation checkpoints.
- **Lineage Tracker:** Persistent species identity, ancestry trees, and mutation logs.
- **Emergence Engine (EPS):** Detect repeated anomalies and propose new UI-integrated laws.
- **Meta-Parameter Interactions:** Synergistic and conflicting effects between laws.
- **Persistence Layer:** Serialize and reload the discovered universe across sessions.
